# Angliška Windows klaviatūra su AltGr Lietuviškais simboliais
### (US English keyboard with AltGr Lithuanian keys for Windows)

Tai angliška–lietuviška klaviatūra Windows sistemai, paremta Linux variantu, kurioje standartiniame 'US English' išdėstyme ir be klaviatūros perjunginėjimų, naudojant AltGr klavišą, pasiekiamos tiek lietuviškos raidės, tiek lietuviškos kabutės ir ilgas brūkšnys (kurių dažnai pasigendama).

Šis išdėstymas pakankamai panašus į Windows sistemos numatytąją lietuvišką klaviatūrą, bet išsprendžia logikos problemas ir įgalina išvengti klaviatūros perjunginėjimų.

Kirčių klausimas nebuvo spręstas.

## Diegimas

1. Atsisiųskite [us-lt-03.zip](https://github.com/ootada/us-lt-03/raw/main/us-lt-03.zip)
2. Išskleiskite archyvą
3. paleiskite `setup.exe`
4. Windows kalbos nustatymuose pridėkite klaviatūrą `US with AltGr Lithuanian keys`
